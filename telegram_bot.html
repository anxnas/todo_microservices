async app.bot.on_shutdown()¶

Выполняется при завершении работы бота.

Закрывает сессию API.

Результат:

None

async app.bot.on_startup()¶

Выполняется при запуске бота.

Инициализирует сессию API, базу данных для локализации и регистрирует диалоги.

Результат:

None

app.bot.register_dialogs()¶

Регистрирует диалоги и маршрутизаторы в диспетчере.

Результат:

None

async app.bot.start_bot()¶

Запускает бота.

Регистрирует функции запуска и завершения, затем запускает поллинг обновлений.

Результат:

None

async app.bot.start_command(message, dialog_manager)¶

Обработчик команды /start.

Проверяет наличие локали пользователя и запускает соответствующий диалог.

Параметры:

  • message – Объект сообщения от пользователя.

  • dialog_manager – Менеджер диалогов для управления состояниями.

Результат:

None

Класс для управления локализацией и пользовательскими настройками языка.

locales¶

Словарь, содержащий локализованные строки для каждого поддерживаемого языка.

Type:

Dict[str, Dict[str, str]]

get_text(key: str, locale: str = 'ru') → str¶

Возвращает локализованный текст для заданного ключа и локали.

Параметры:

  • key (str) – Ключ локализованной строки.

  • locale (str, optional) – Код языка. По умолчанию „ru“.

Результат:

Локализованная строка или исходный ключ, если строка не найдена.

Тип результата:

str

async get_user_locale(telegram_id: int) → str¶

Получает предпочитаемый язык пользователя.

Параметры:

telegram_id (int) – ID пользователя в Telegram.

Результат:

Код языка пользователя или „ru“, если язык не установлен.

Тип результата:

str

async init_db() → None¶

Инициализирует базу данных для хранения пользовательских настроек языка. Создает базу данных и таблицу telegram_user, если они не существуют.

load_locales() → None¶

Загружает локализованные строки из JSON-файлов для русского и английского языков.

async set_user_locale(telegram_id: int, locale: str) → None¶

Устанавливает предпочитаемый язык для пользователя.

Параметры:

  • telegram_id (int) – ID пользователя в Telegram.

  • locale (str) – Код языка для установки.
